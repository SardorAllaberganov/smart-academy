Создание игр в Sratch

Ваш ребёнок много времени проводит играя за компьютером? Как превратить это время в пользу?

Игры на компьютере способны привить ребенку интерес к современным IT- технологиям и программированию. Хотите, чтобы ваш ребенок превратился из потребителя технологий в их создателя ?

Тогда запишите его на курс по созданию компьютерных игр.

Scratch- это понятный и увлекательный язык программирования, который разработан Массачусетским технологическим университетом специально для детей. С помощью Scratch программирование, ваш ребенок сделает первые шаги в мир алгоритмов, команд и роботов. Занятия проводятся в интересной игровой форме по методике основанной на простом и эффективном языке визуального программирования - Scratch.

Пройдя курс Scratch для детей, ваш ребенок:

  • познакомиться с основами программирования с помощью визуального языка программирования Scratch;
  • научиться полностью взаимодействовать с различными объектами, перемещать их по экрану, видоизменять и играть с ними, давать определенные функции объектам для дальнейших действий;
  • создавать собственные анимированные и интерактивные истории, презентации, модели, игры и другие произведения;
  • углубит знание математики;
  • разовьет алгоритмические мышление;
  • научит работать в команде;

По окончанию курса каждый ребенок получит базовые практические навыки программирования, необходимые для изучения любых других языков программирования, таких как Python, PHP, Java или JavaScript. Программа составляется с помощью разноцветных визуальных блоков-кирпичиков, которые нужно просто перетаскивать и соединять, как кубики в конструкторе Lego. В программу нужно вносить изменения в любой момент-результат виден сразу.

Как установить Scratch на домашний компьютер:

  1. Перейдите в раздел скачивания на официальном сайте: https://scratch.mit.edu/download
  2. После установите скачанный файл Scratch
  • В Windows нужно установить приложение и запустить его
  • В Mac OS нужно перетащить иконку приложения из образа в папку с приложениями запустить его
  • В Linux можно скачать и установить файл или использовать << via PPA >>
